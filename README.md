# 🧠 Qt AI Debug Assistant

An intelligent Qt-based desktop application that transforms raw logs into actionable insights using AI-like rule-based analysis and interactive UI feedback.

---

## 🚀 Features

✅ Smart log analysis (Error / Warning detection)  
✅ AI-like rule engine for pattern matching  
✅ Context-aware suggestions  
✅ Smart popup assistant for critical issues  
✅ Clean and modern Qt GUI  
✅ Highlighted issues for better readability  

---

## 💡 How It Works

1. Load or paste logs into the application  
2. System preprocesses and splits log data  
3. AI-like rule engine analyzes patterns  
4. Issues are detected and categorized  
5. Suggestions are generated automatically  
6. Critical alerts are shown via popup assistant
